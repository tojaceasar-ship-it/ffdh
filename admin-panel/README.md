# FFDH Admin Panel

Admin dashboard for managing Fruits From Da Hood e-commerce operations.

## Features

- Order management
- Customer analytics
- Product inventory
- Content management
- User administration

## Setup

```bash
cd admin-panel
npm install
npm run dev
```

## Deployment

This admin panel is deployed separately from the main application for security reasons.

## Access

Requires Supabase authentication with admin role.
